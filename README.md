# Indian Music Website

## Project Overview

This project is a multi-page website about Indian classical music, created as part of a Web Design assignment. The website is built using only **HTML5** and **CSS3**, demonstrating fundamental web development concepts such as page structure, navigation, styling, and content organization.

The website introduces visitors to the rich traditions of Indian music, including Hindustani and Carnatic classical music, along with information about other musical forms.

Website is published in Github pages, link here: https://jmichael2025.github.io/WebDesign/webdesign/index.html
---

## Technologies Used

* HTML5
* CSS3

No JavaScript, frameworks, or external libraries were used in this project.

---

## Website Structure

The website consists of five HTML pages:

| Page       | Description                                             |
| ---------- | ------------------------------------------------------- |
| Home       | Introduction to Indian Music                            |
| About      | Information about the website and Indian music heritage |
| Hindustani | Overview of Hindustani Classical Music                  |
| Carnatic   | Overview of Carnatic Classical Music                    |
| Other      | Other forms of Indian music and musical traditions      |

## How to Run

1. Download or clone the project files.
2. Ensure all HTML files and `style.css` are in the same folder.
3. Open `index.html` in any modern web browser.
4. Navigate through the website using the menu links.
